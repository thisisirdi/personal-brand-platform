# Roadmap

## V1

- Public personal brand site
- Home, About, Experience, Projects, Project Detail, Contact
- Database-managed projects
- Contact form validation and persistence
- Prisma schema and seed data
- Documentation for architecture, data model, roadmap, and analytics future

## V2

- Admin workflow for project management
- Supabase Auth for protected admin access
- Blog or builder notes
- Draft and preview states for content
- Better contact operations such as email notifications or CRM export
- Real project images and richer case study media

## V3

- Dashboard for content and contact activity
- PostHog implementation for product analytics
- Content performance reporting
- Gallery or creative experiment archive
- More structured project outcomes and metrics

## Explicitly Deferred

- Authentication in V1
- PostHog implementation in V1
- Blog in V1
- Services or freelance page
